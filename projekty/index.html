Screeps server list @ screeps.fordo.sk
SCreeps je menej znama hra. Je to strategia kde sa stavaju budovy a panacikovia. Rozdiel je ale v tom ze sa to neovlada pomocou klavesnice a mysi, ale pomocou scriptovania v Node.JS . Existuje officialny server ktory je plateny. A potom su rozne sukromne. takzvane private servere. Su zdarma ale maju trochu horsie kvalitu zvycajne. Najst ich je ale celkom narocne. Takze som spravil web kde ludia mozu pridat svoj server, bude na nom zobrazene ci je aktualne online, a kolko je na nom hracov.

TeamSpeak3 hosting @ ts3.fordo.sk
TeamSpeak3 hosting @ ts3.fordo.sk Mat vlastny TeamSpeak3 server je super vec. Nemusite sa s nikim delit, a nikto vas nebude vyhazdzovat. Takze napad spravit si vlastny server je jasny. Videl som ale prilezitost spravit server aj pre par znamych ludi vo svojom okoli. Z tohto napadu sa rozrastlo na napad spravit na to automat kde si ludia mozu spravit svoj vlastny kedykolvek. Spravit web nebola ziadna raketova veda. A nejaky script ktory vytoci kontajner s TeamSpeak-om tiez nebolo zas tak narocne.
