Ako na HTTPS s Lets Enrypt

Jan 1, 0001 Návod ako rozbehať zabazpečený protokol HTTPS na web server apache2 na Debian.

Nainštalujeme nástroj:

sudo apt install python-certbot-apache

Spustíme ho, vybereme ktorý virtual host chceme použiť, musíme mať len jeden v každom súbore

certbot --authenticator webroot --installer apache

Tiež musíme zadať cestu do DocumentRoot adresára, preto pokiaľ používame virtualhost iba na Redirect alebo Proxy, musíme to na chvíľu vypnúť a nastaviť ho. Po skončení skriptu môžme daľej pokračovať v používaní predošlej konfigurácie, s tým čo nam pridal skript.

Pokiaľ nám prehliadač stále píše že pripojenie nieje plne zabezpečené, je potreba pozrieť všetky linky, niekde je asi použitá plná cesta s http:// .

Alias /.well-known/acme-challenge/ /var/www/domena/web/acme-challenge/
<Directory /var/www/domena/web/acme-challenge/>
AllowOverride None
Require all granted
</Directory>

ProxyPass "/.well-known/acme-challenge/" !\
