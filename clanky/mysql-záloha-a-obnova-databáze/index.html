MySQL - záloha a obnova databáze

Jan 1, 0001 Príkazy ako spraviť MySQL zálohu databáze, a obnovenie databáze zo zálohy.

Vyzálohovanie vsštkých databáz:

/usr/bin/mysqldump --all-databases | gzip -c -9 > /opt/backup/sql/dump_$(date '+%Y-%m-%d').sql.gz

Nahranie zálohy na server:

mysql -h $SERVER -u $USER -p < "dump_"$DATUM".sql.gz"

Aby to fungovalo, musíme ešte do MySQL konzole zadať príkaz na spracovanie privilegií:

FLUSH PRIVILEGES
