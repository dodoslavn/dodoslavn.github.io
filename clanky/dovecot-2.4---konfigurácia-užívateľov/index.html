Dovecot 2.4 - Konfigurácia užívateľov

8. December 2025 • Dodoslav
dovecot mail imap
Ako nastaviť používateľov, mená a heslá pre nový Dovecot na Debiane. Pôjde len o statický súbor.

Výber autentifikácie

Zakomentovať alebo vybrať jeden správny spôsob autentifikácie v /etc/dovecot-conf.d/10-auth.conf

#!include auth-deny.conf.ext
#!include auth-master.conf.ext
#!include auth-oauth2.conf.ext

#!include auth-system.conf.ext
#!include auth-sql.conf.ext
#!include auth-ldap.conf.ext
#!include auth-passwdfile.conf.ext
!include auth-static.conf.ext\

Nastavenie autentifikácie

Konfigurácia statického súbora /etc/dovecot-conf.d/conf.d/auth-static.conf.ext

auth_mechanisms = plain login
auth_allow_cleartext = no

passdb passwd-file {
passwd_file_path = /etc/dovecot/passdb
# passdb_default_password_scheme = SHA512-CRYPT
}

userdb static {
fields {
uid = vmail
gid = vmail
home = /var/vmail/%{user}
mail_driver = maildir
mail_path = /var/vmail/%{user}/Maildir # full mail location
}
}\

Vygenerovanie hesiel

Hash hesla sa dá vygenerovať pomocou

doveadm pw -s SHA512-CRYPT

Nastavenie hesiel a účtov

Vygenerované hash-e následne vložíme do statického súbora

cat /etc/dovecot/passdb
