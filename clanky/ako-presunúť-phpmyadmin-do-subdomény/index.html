Ako presunúť PHPMyAdmin do subdomény

17. September 2017 • Dodoslav
phpmyadmin apache2 php mysql linux debian
Návod ako presunúť tento nástroj napríklad z fordo.sk/phpmyadmin do subodmény phpmyadmin.fordo.sk .

Vypneme aktuálny virtualhost:

sudo a2disconf phpmyadmin

Vytvoríme nový PHPMyAdmin konfiguračný súbor virtualhostu:

vi /etc/apache2/sites-enabled/new_phpmyadmin.conf

Vložíme doň obsah súboru phpmyadmin.conf{target="_new"} a upravíme doménu podľa seba.

Povolíme ho:

sudo a2enconf

Reštartujeme apache2

sudo systemctl reload apache2.service
