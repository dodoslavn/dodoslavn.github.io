Event okno na Windows Server 2003

13. September 2011 • Dodoslav
server windows win 2003 03 event okno Group Policy
Ak ste si pred chvíľou nainštalovali Windows Server 2003 a hľadáte riešenie na vypnutie tzv. Event okna ktorá sa vás neustále pýta otravné a takmer nevyužiteľné otázky ako: prečo vypínate počítač, tak tento návod je práve pre vás!

Celé je to veľmi jednoduchá a rýchla vec, takže bez zbytočných obkecov ideme na to!

Takže najskôr si otvoríme takzvaný:

Group Policy Object Editor

Ten spustíme tak že pôjdeme na:

START -> RUN / SPUSTIT

a tam napíšeme:

gpedit.msc

..a potvrdíme. Objavilo sa nám okno, a teraz na paneli (na ľavej strane) sa budeme postupne preklikávať:

Computer Configuration -> Administrative Templates -> System

Na pravej strane máme zoznam položiek, v kt. musíme nájsť:

Display Shutdown Event Tracker

Po dvoj-kliku na túto položku sa nám otvorí okno v ktorom označíme:

Disabled

Celé to potvrdíme tlačítkom OK a už pri vypínaní počítača žiadne dalšie okno nebude.
