Záloha registra na Windows XP

Jan 1, 0001 Ak ste strávili vyše času pri ladení svojho Windowsu, určite nechcete ani vidieť niečo ako BSoD či aj niečo horšie. V tomto článku si ukážeme ako si uložiť svoje nastavenie registra.

Stačí si spustiť Windowsovu utilitu Backup

START -> ALL PROGRAMS -> ACCESSORIES -> SYSTEM TOOLS -> BACKUP

alebo kto to má po slovensky tak

START -> PROGRAMY -> PRISLUSENSTVO -> SYSTEMOVE NASTROJE -> BACK UP

Takato záloha registru síce zaberá niekedy 500MB niekedy viacej. To záleží ako je preplnený. Takúto zálohu si môžete potom aj napáliť na disk a pri instalácií nového Windowsu len cez Reagedit importovať.

A teraz samotný návod:

KROK 1 - Spustíme spomínanú utilitu.

KROK 2 - Ked sa nám spustí "Sprievodca" tak klikneme na "Advenced mode" a kto to má po slovensky tak "Spresniť".

KROK 3 - Otvoríme vetvu nášho počítača, a tam sa nám zobrazia disky, mechaniky.. a nakonci je "System state" a kto to má po slovensky tak zase "Systém štátu" a oznacíme. Napravo sa nam zobrazí aj náš register. Dolé si klikneme kde to chceme ulozit.

KROK 4 - Daľej už len klikneme na tlačítko "Start backup"
