Linux inštalácia youtube-dl

20. August 2016 • Dodoslav
linux debian youtube youtube-dl
Ako správne nainštalovať stahovač YouTube videí, youtube-dl. Na Debian 8 (jessie) sa to musí robiť trochu inak z dôvodu že v repozitároch ma starú verziu.

Inštalácia:

sudo apt install youtube-dl

Kontrola nainštalovanej verzie:

youtube-dl --version

V repozitároch môže byť ale často verzia stará, preto existuej alternatívny spôsob:

wget -O - https://yt-dl.org/downloads/latest/youtube-dl | sudo tee /usr/sbin/youtube-dl >/dev/null sudo chmod a+x /usr/sbin/youtube-dl

Skontrolujeme že máme novšiu verziu:

/usr/sbin/youtube-dl --version

Pár príkladov použitia:

Príkaz na stiahnute a automatické prekonverovanie YouTUbe videa do mp3

youtube-dl --extract-audio --audio-quality 0 --audio-format mp3 -ti https://www.youtube.com/watch?v=XXX

Ziskanie nazvov videi z playlistu:

youtube-dl --ignore-errors -o "%(title)s.%(ext)s" --get-filename https://www.youtube.com/playlist?list=XXX

Ine:

youtube-dl -o "$PATH_DL%(uploader)s/mp4/%(playlist)s/%(title)s.%(ext)s" --download-archive archive.txt --format best --ignore-errors --continue --no-overwrites --verbose -a $PATH_BASE""links/playlists.csv
