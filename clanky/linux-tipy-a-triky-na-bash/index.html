Linux - tipy a triky na Bash

Jan 1, 0001 Zbierka tipov a trikov na Linuxový Bash

Vysvetlenie príkazu stat

There are 3 kind of "timestamps":
Access - the last time the file was read
Modify - the last time the file was modified (content has been modified)
Change - the last time meta data of the file was changed (e.g. permissions)\

Presunitie sa v skripte do adresára skriptu:

cd "$(dirname "$0")"

Získanie názvu skriptu v skripte samotnom:

echo $(basename "$0")

Prehľadávanie súbora po riadkoch. Premenná IFS (Internal Field Separator) sa použije na nastavenie znaku na základe ktorého sa obsah súboru rozseká do premennej v cykle typu for .

IFS=$'\n'
for RIADOK in $(cat subor.txt)
do
echo $RIADOK
done\

hladanie pipe: zistanie ci je subor otvoreny ls -l /proc/*/fd/* 2>/dev/null | grep FILENAME zistenie PID: ps -ef | grep process_name ls -l /proc/$PID/fd/0 ls -l /proc/*/fd/1 | grep 47673884 ps -ef | grep $NEW_PID
