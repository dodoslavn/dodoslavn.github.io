Zabbix monitoring - manuál ako na to

Jan 1, 0001 Návod ako nainštalovať Zabbix monitoring na Linux, aj do kontajnera.

Návod pre Debian 9, prípadne Debian 9 v Dockeri.

Priprava

Otvoríme v prehliadači tento odkaz kde si nájdeme akyuálnu verziu balíku, a skopríjeme jeho odkaz. Balík následne na servere stiahneme cez wget. https://repo.zabbix.com/zabbix/4.2/debian/pool/main/z/zabbix-release/

Inštalácia základných vecí.

apt install dialog apt-utils wget vim

Stiahneme Zabbix balík:

wget https://repo.zabbix.com/zabbix/4.2/debian/pool/main/z/zabbix-release/zabbix-release_4.2-1%2Bstretch_all.deb

Nainštalujeme ho:

dpkg -i zabbix-release_4.2-1+stretch_all.deb

Obnovíme zoznam balíkov:

apt update

Server

Inštalácia nástrojov na monitoring server.

Inštalácia MariaDB servera:

apt install mysql-server mysql-client

Spustenie databázového servera:

/etc/init.d/mysql restart

Inštalácia Apache2 web servera:

apt install apache2

Inštalácia PHP a webového nástroja do Apache web servera, na spravovanie MariaDB databáze:

apt install phpmyadmin

Spustenie Apache2 web servera:

/etc/init.d/apache2 restart

Inštalácia samotného monitoring servera:

apt install zabbix-server-mysql

Otvoríme v prehliadači PHP My Admin ktorý sme naištalovali. Prihlasovacie údaje použijeme tie čo sme zadali pri inštalácií. Mal by byť na adrese: http://localhost/phpmyadmin/

Následne v PHP My Admin vytvoríme nového užívateľa. Pri vytvárani

Inštalácia Zabbix web-u:

apt install zabbix-frontend-php

rm /var/www/html/index.html
cp -r /usr/share/zabbix/* /var/www/html/

Nastavenie agenta

Hostname=DEVMDMAPP01

Server=zabbix_server ServerActive=zabbix_server
