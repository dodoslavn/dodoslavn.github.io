Frekvenčné pásma

29. July 2024 • Dodoslav
rtl sdr rtl-sdr noaa satelite frekvencie pasmo
Zoznam všetkých frekvenčných pásiem, názov aj s frekvenciou.

::: {style=“margin-bottom: 1em;”}

1. Extrémne nízka frekvencia (ELF)

Frekvenčné pásmo: 3 Hz až 30 Hz

Použitie: Komunikácia so ponorkami, geologické prieskumy. :::

::: {style=“margin-bottom: 1em;”}

2. Super nízka frekvencia (SLF)

Frekvenčné pásmo: 30 Hz až 300 Hz

Použitie: Komunikácia so ponorkami, špecifické aplikácie v oblasti komunikácie. :::

::: {style=“margin-bottom: 1em;”}

3. Ultra nízka frekvencia (ULF)

Frekvenčné pásmo: 300 Hz až 3 kHz

Použitie: Komunikácia so ponorkami, geologické prieskumy, niektoré vojenské aplikácie. :::

::: {style=“margin-bottom: 1em;”}

4. Veľmi nízka frekvencia (VLF)

Frekvenčné pásmo: 3 kHz až 30 kHz

Použitie: Komunikácia so ponorkami, navigácia, časové signály. :::

::: {style=“margin-bottom: 1em;”}

5. Nízka frekvencia (LF)

Frekvenčné pásmo: 30 kHz až 300 kHz

Použitie: AM rádio vysielanie, navigácia, časové signály. :::

::: {style=“margin-bottom: 1em;”}

6. Stredná frekvencia (MF)

Frekvenčné pásmo: 300 kHz až 3 MHz

Použitie: AM rádio vysielanie, námorná komunikácia, letecká navigácia. :::

::: {style=“margin-bottom: 1em;”}

7. Vysoká frekvencia (HF)

Frekvenčné pásmo: 3 MHz až 30 MHz

Použitie: Krátkovlnné rádio vysielanie, amatérske rádio, medzinárodné vysielanie, námorná a letecká komunikácia. :::

::: {style=“margin-bottom: 1em;”}

8. Veľmi vysoká frekvencia (VHF)

Frekvenčné pásmo: 30 MHz až 300 MHz

Použitie: FM rádio vysielanie, televízne vysielanie (analógové a digitálne), VHF letecká pásma (aviácia), námorná komunikácia, komunikácia v oblasti verejnej bezpečnosti. :::

::: {style=“margin-bottom: 1em;”}

9. Ultra vysoká frekvencia (UHF)

Frekvenčné pásmo: 300 MHz až 3 GHz

Použitie: Televízne vysielanie (digitálne), mobilné telefóny (vrátane LTE), GPS, Wi-Fi, niektoré radarové systémy. :::

::: {style=“margin-bottom: 1em;”}

10. Super vysoká frekvencia (SHF)

Frekvenčné pásmo: 3 GHz až 30 GHz

Použitie: Satelitná komunikácia, mikrovlnné rádiové relé, radar (počasie a vojenský), Wi-Fi (5 GHz pásmo), niektoré vedecké aplikácie. :::

::: {style=“margin-bottom: 1em;”}

11. Extrémne vysoká frekvencia (EHF)

Frekvenčné pásmo: 30 GHz až 300 GHz

Použitie: Radar s vysokým rozlíšením, satelitná komunikácia, millimetrová radarová technológia, niektoré vedecké aplikácie a pokročilé komunikačné technológie (napríklad 5G). :::

::: {style=“margin-bottom: 1em;”}

12. Terahertzové pásmo (THz)

Frekvenčné pásmo: 300 GHz až 3 THz

Použitie: Pokročilé zobrazovacie systémy, bezpečnostné skenovanie, vedecký výskum a novú komunikačnú technológiu. :::
