Yum - tipy a triky

Jan 1, 0001 Nejake tipy a triky an yum. Pre CentOS a RedHat.

Rozbity repozitar

Ako fixnut rozbity repozitar yum-u. Stava sa to celkom casto.

mkdir /var/lib/rpm/backup
cp -a /var/lib/rpm/__db* /var/lib/rpm/backup/

/var/lib/rpm/__db
rm -f /var/lib/rpm/__db.[0-9][0-9][0-9]
rpm --quiet -qa

rpm --rebuilddb
yum clean all

Verzie

Jak vylistovat vsetky verzie nejakeho baliku.

yum -v list package_name --show-duplicates

Balíky vs. súbory

Ako zistiť aký balík mi nainštaluje konkrétny súbor.

rpm -q --whatprovides package_name

yum whatprovides package_name

Ako zistiť aké súbory mi nainštaluje konkrétny balík.

rpm -ql package_name

Ako získať obsah RPM balíka

rpm2cpio balik.rpm | cpio -idmv
