Inštalácia MySQL servera na Debian 9

Jan 1, 0001 V repozitároch Debian 9 sa už MySQL nenachádza z dovôdu že bol nahradený MariaDB. To ale neznamená že ho nemôžme používať a nainštalovať mimo repozitár.

Najskôr musíme stiahnuť officiálny balík zo stránok MySQL ktorý nám nakonfiguruje repozitár. Buď navštívime stáránku a stiahneme najnovší z https://dev.mysql.com/downloads/repo/apt/{target="_new"} .

Alebo tu je link na balík z doby písania článku:

wget https://dev.mysql.com/get/mysql-apt-config_0.8.7-1_all.deb

Po stiahnutí balíka ho môžme nainštalovať:

sudo dpkg -i mysql-apt-config_0.8.7-1_all.deb

Po inštalácií spravíme update, v ktorom by sme mali vidieť mysql repozitár:

sudo apt update

A teraz uz môžme nainštalovať samotný MySQL server priamo z repozitára:

sudo apt install mysql-server

Po ňom môžme nainštalovať napríklad PHPMyAdmin:

sudo apt install phpmyadmin
