Linux a Dual Boot

Jan 1, 0001 V tomto článku si ukážeme ako nastaviť populárny DualBoot, čo znamená že pred bootovaním operačného systému sa nás zavádzač opýta aký operačný systém chceme spustiť. Všetky spomenuté spôsoby fungujú iba s MBR a BIOS. Nefungujú preto s UEFI.

Je niekoľko spôsobov ako to dosiahnuť. Pokiaľ máme nainštalovaný Linux ale spúšta sa nám iba priamo MS Windows, nabootujeme nejaku Live distribúciu z USB alebo cez CD/DVD-ROM. Osobne používam Ubuntu. Na verzi nezáleží, takže môžme použiť aj staršiu inštalačku, musíme však dodržať architektúru.

Po nabootovaní Linuxu otvoríme terminál a pripojime disk kde máme uz nainštalovaný Linux do "/mnt".

sudo mount /dev/sda1 /mnt ;

Nabindujeme pár potrebných adresárov

sudo mount --bind /dev /mnt/dev ;

sudo mount --bind /sys /mnt/sys ;

sudo mount --bind /proc /mnt/proc ;

A môžme zkúsiť tzn. Change Root.

sudo chroot /mnt ;

Pokial bol uspešný, mali by sme byť prepnutý v nainštalovanom Linuxe, môžme teda zkúsiť nainštalovať GRUB znova.

update-grub ;

grub-install /dev/sda ;

grub-install --recheck /dev/sda ;




\

Alebo ešte jeden spôsob ktorý kedysi fungoval

sudo fdisk -l

sudo mount /dev/sda1 /mnt

sudo grub-install --root-directory=/mnt /dev/sda

sudo update-grub
