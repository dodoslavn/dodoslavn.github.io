Unbound DNS resolver na Debian-e 13

27. January 2026 • Dodoslav
dns resolver cache unbound debian
Ako nainštalovať DNS resolver Unbound na Debian 13.

Ako nainštalovať DNS-cache server na Debian, konkrétne Unbound.

Inštalácia

apt install bind9-dnsutils net-tools dns-root-data iproute2
apt install unbound

Nastavenie ovládania

Remote-control tu vypínam, zakomentované riadky niesú podporované v tejto verzií Unbound.

echo 'remote-control:
   control-enable: no
   control-interface: /run/unbound.socket
   #control-use-cert: yes
   #control-perms: 600
   #control-owner: root
   #control-group: root' > /etc/unbound/unbound.conf.d/remote-control.conf

Základné nastavenia

Tu si dať pozor na ten access-control a pridať tam svoju sieť ak je iná.

echo ' server:
   interface: 0.0.0.0

   interface: ::0

   access-control: 127.0.0.0/8 allow
   access-control: 192.168.1.0/24 allow

   port: 53
   do-ip4: yes
   do-udp: yes
   do-tcp: yes

   verbosity: 1' > /etc/unbound/unbound.conf.d/lan.conf

Overenie funkčnosti

Reštartujeme Unbound a skontrolujeme či beží ako služba:

systemctl restart unbound.service
systemctl status unbound.service

Overíme či naozaj prekladá domény:

dig google.sk @127.0.0.1
