Samba server na Debian

Jan 1, 0001 Ako nainštalovať službu pre zdielanie súborov z Linuxu na Windows pomocou Samba .

Porty to používa 139 a 445.

Inštalácia

apt install samba

Konfigurácia

vi /etc/samba/smb.conf

Príklad

[share]
read only = no
path = /data
guest ok = yes
browsable = yes\
