Virtualbox - inštalácia Guest additions

17. July 2019 • Dodoslav
virtualbox
Návod ako stiahnuť a nainštalovať Virtualbox Guest additions do Debian virtuálky.

Inštalácia potrebných OS balíkov

apt install build-essential module-assistant dkms make

m-a prepare

Stiahnutie

VBOXS=$(vboxmanage -v | cut -d'r' -f1)
VBOXL=$(vboxmanage -v | sed 's/r/-/' )
wget http://download.virtualbox.org/virtualbox/"$VBOXS"/VirtualBox-"$VBOXL"-Linux_amd64.run
