NtopNG - inštalácia na Debian

2. September 2024 • Dodoslav
ntop ntopng network security
Ako nainštalovať Ntopng na Debian.

Stiahnutie balíka

$ wget https://packages.ntop.org/apt/$(lsb_release -cs)/all/apt-ntop.deb -O /tmp/apt-ntop.deb

Inštalácia repozitára

# apt install /tmp/apt-ntop.deb
# apt update

Inštalácia

# apt install ntopng

Nastavenie

Vytvoríme nový konfiguračný súbor v zložke kde sa načítavajú konfigurácie automaticky.

# vi /etc/ntopng/ntopng.conf.d/my_own.conf

Obsah môže vyzerať napríklad takto, pozor - port má rovnaký ako používa Grafana.

# port
-w=3001

# interface
-i=enp8s0

#
#--local-networks "192.168.0.0/24"

Zapnúť službu:

# systemctl restart ntopng.service

Proxy

Pristupovať na webové rozhranie cez Apache Reverse-Proxy je jednoduché, a nevyžaduje žiadne špeciálne pravidlá.

Prenastavenie dátovej zložky do RAM drive

Vytvoríme zložku pomocou SystemD

echo 'd /tmp/ntopng 1777 ntopng ntop -' > /etc/tmpfiles.d/ramdrive_ntopng.conf

Pre aplikovanie hned teraz

systemd-tmpfiles --create /etc/tmpfiles.d/ramdrive_ntopng.conf

Otvoríme fstab súbor na automatické vytvorenie dočasného diskového oddielu

# editor /etc/fstab

Vložíme riadok

tmpfs /tmp/ntopng tmpfs size=500M,mode=1777 0 0

Pre aplikovanie hned teraz

systemctl daemon-reload

Nastavíme symlink do nového dočasného diskového oddielu

ln -s /tmp/ntopng/ /var/lib/ntopng

Prípadne sa dá nastaviť cesta k dočasnému disku priamo v Ntopng

echo '--data-dir=/tmp/ntopng' >> /etc/ntopng/ntopng.conf.d/my_own.conf
