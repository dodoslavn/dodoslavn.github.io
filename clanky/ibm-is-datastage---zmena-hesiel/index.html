IBM IS Datastage - zmena hesiel

14. January 2019 • Dodoslav
ibm datastage
Manuál ako zmeniť všetky heslá na IBM IS Datastage

Note: this is procedure to update password for OS accounts which are used by Datastage.

User "xmeta":

KB link:
https://www.ibm.com/support/knowledgecenter/en/SSYQ72_10.0.4/com.ibm.help.suiteinstall1004.reportconfig.doc/T_ResettingPasswordForXMetaAndCognos.html
1.) switch to main DS user
su - dsadm
2.) move to the folder
cd /opt/IBM/InformationServer/IIS115/ASBServer/bin

3.) update the password via script
./AppServerAdmin.sh -db -user xmeta -password

4.) verify the password via command
./AppServerAdmin.sh -w


User "xmetasr":
KB link: http://www-01.ibm.com/support/docview.wss?uid=swg21640075

1.) switch to main DS user
su - dsadm

2.) move to the folder
cd /opt/IBM/InformationServer/IIS115/ASBServer/bin

3.) update the password via script
./AppServerAdmin.sh -dbs -user xmetasr -password

3.) update the password via script (script doesnt existed)
./model_registry_synchronize.sh -l


User "dsodb":
KB link:
https://www.ibm.com/support/knowledgecenter/en/SSZJPZ_8.7.0/com.ibm.swg.im.iis.ds.monitor.doc/topics/changingpassword.html

1.) switch to main DS user
su - dsadm
2.) stop Datastage engine

3.) move to the folder

cd /opt/IBM/InformationServer/IIS115/ASBNode/bin

4.) encrypt your new password
./encrypt.sh

5.) move to the folder
cd /opt/IBM/InformationServer/IIS115/Server/DSODB

6.) update the pass phrase (which you have from step 4. ), in this script in variable Password

vi DSODBConnect.cfg
User "dsadm":

KB link: http://www-01.ibm.com/support/docview.wss?uid=swg21426789\
