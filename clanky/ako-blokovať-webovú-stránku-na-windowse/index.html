Ako blokovať webovú stránku na Windowse

17. August 2011 • Dodoslav
windows blokovanie web stranka
Tu si ukážeme ako veľmi jednoducho a účinne, zablokovať webovú stránku na operačnom systéme Windows. Tiež si ukážeme aj spôsoby ako sa to dá, no to už len okrajovo, pretože sa môžu veľmi líšit vzávislosti aké zariadenia máte.

Takže bez zpytočných obkecov začneme.
Presunieme sa na adresár:

C:\WINDOWS\system32\drivers\etc

V tomto adresáre sa nachádza súbor

hosts

Otvoríme ho v nejakom textovom editore ako napríklad Notepad alebo aj v Poznámkovom bloku. Do tohto súboru budeme na úplny spodok vkladať domény stránok ktoré chceme zabokovať. Vždy na nový riadok napíšeme svoju IP adresu localhostu. Tá je vždy:

127.0.0.1

a o medzeru(na velkosti medzere nezáleží) za nu napíšeme jednu doménu stránky ktorú chceme zablokovať. No ak by sme chceli zablokokovať napr. prístup na Pokec, nestačí zadať iba stránku pokec.sk, ale všetky možné spôsoby, kedže na Pokec sa da pristupovať rôznymy doménamy(aj ked konečná doména je vždy rovnaká). Takže musíme zadať aj: azet.sk, pokec.azet.sk ale aj kombinácie s www . Ked sme pri tom Pokeci, ukážeme si ako môže vyzerať konečný súbor.

# Copyright (c) 1993-1999 Microsoft Corp.
#
# This is a sample HOSTS file used by Microsoft TCP/IP for Windows.
#
# This file contains the mappings of IP addresses to host names. Each
# entry should be kept on an individual line. The IP address should
# be placed in the first column followed by the corresponding host
# name. The IP address and the host name should be separated by
# at least one
# space.
#
# Additionally, comments (such as these) may be inserted on individual
# lines or following the machine name denoted by a '#' symbol.
#
# For example:
#
# 102.54.94.97 rhino.acme.com # source server
# 38.25.63.10 x.acme.com # x client host


127.0.0.1 www.pokec.sk
127.0.0.1 pokec.sk
127.0.0.1 azet.sk
127.0.0.1 pokec.azet.sk

\

Všetko čo je v tomto súbore napísané za # je iba komentár a celí riadok za # bude ignorovaný. No chcel by som ešte niečo dodať ohladom toho, čo vlastne tento súbor robí. Pretože on neslúži na blokovanie stránok ale na presmerovanie domeny na IP adresu, kedze domena je vlastne len náhrada za nu. Takže v skutočnosti sme stránku nezablokovali ale iba presmerovali na váš počítač. Ale kedže na Vašom počítači nemáte nainštalovaný Apache alebo iny webový server, stránka sa nezobrazí. Pokial máte nainštalovaný, tak stačí v nom nastaviť aby na tieto domény nereagoval, pripadne nasmeroval niekam kde by sa nič nezobarzilo.
