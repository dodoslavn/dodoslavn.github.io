Inštalácia Airflow

13. February 2025 • Dodoslav
etl databaza database db sql etl bi dwh airflow
Postup ako zprovozniť Airflow - ETL nástroj od Apache.

Inštalácia systémových balíkov

apt install -y default-libmysqlclient-dev build-essential

Inštalácia samotného Airflow

cd /niekam/ python3 -m venv . ./bin/pip3 install apache-airflow[mysql] ./airflow db init

Úprava konfigurácie

Otvoríme súbor

vi ../home/airflow/airflow.cfg

a nastavíme v nom

sql_alchemy_conn = mysql+mysqldb://airflow_user:your_secure_password@localhost:3306/airflow_db executor = LocalExecutor

Inicializácia databáze

./airflow db init

Pridanie účtu

./airflow users create your_username --firstname first_name --lastname last_name --role Admin --email username@example.com
