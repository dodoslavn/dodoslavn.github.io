NtopNG - inštalácia na Debian

Mar 8, 2025 Ako nainštalovať Ntopng na Debian.

Stiahnutie balíka $ wget https://packages.ntop.org/apt/$(lsb_release -cs)/all/apt-ntop.deb -O /tmp/apt-ntop.deb Inštalácia repozitára

apt install /tmp/apt-ntop.deb

apt update

Inštalácia

apt install ntopng

Nastavenie Vytvoríme nový konfiguračný súbor v zložke kde sa načítavajú konfigurácie automaticky.

vi /etc/ntopng/ntopng.conf.d/my_own.conf

Obsah môže vyzerať napríklad takto, pozor - port má rovnaký ako používa Grafana.

port

-w=3001

interface

-i=enp8s0

#–local-networks “192.168.0.0/24” Zapnúť službu:

systemctl restart ntopng.service
